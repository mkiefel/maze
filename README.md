Simple Maze for C++
===================

Dependencies
------------
- [CImg.h](http://cimg.sourceforge.net/)
- C++11
- [libpng](http://www.libpng.org/pub/png/libpng.html)

Build
-----
    $ mkdir build && cd build
    $ cmake ..
    $ make

Run
---
Still in the same directory you can run the code with

    $ ./maze/builder out.png
